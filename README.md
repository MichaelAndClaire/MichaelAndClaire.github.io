# MichaelAndClaire.github.io

Wedding website for Michael and Claire, built with Astro, Tailwind CSS, Formspree, and GitHub Pages.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production site deploys to `https://michaelandclaire.github.io` through GitHub Actions.

Most wedding details live in `src/data/wedding.ts`.
